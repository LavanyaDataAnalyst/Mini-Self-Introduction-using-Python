# Basic Self Intro Creator – Python Project
## ℹ️ Description:
* The program takes details such as name, age, city, education, and skills from the user and displays them in a structured format. 
* It demonstrates fundamental programming concepts in Python.

## 📖 Table of Contents: 
>* Project Overview
>* Features
>* Tools & Technologies
>* Sample output
>* How to Use

## 🔍 Project Overview

* This project is a simple Python-based application that collects user input and generates a formatted student profile.
* It helps beginners understand input handling, string formatting, and basic Python operations.

## ⚙️ Features

* Accepts user input dynamically
* Stores and processes data using variables
* Formats output in a readable structure
* Beginner-friendly Python logic

## 🛠️ Tools & Technologies

* Python – Core programming language
* Jupyter Notebook / VS Code – Development environment

## 💡 Sample Output

```
---------- My Intro ----------
My name is John
I am 20 years old
I am currently living in Chennai
I hold BSc degree
I am currently pursuing Data Analytics
I have developed my skills in Excel, SQL
Now I am currently learning Python
```

## ▶️ How to Use

* 📜 How to Run
* Install Python on your system
* Open the .ipynb file in Jupyter Notebook or VS Code
* Run all cells
* Enter the required details when prompted
* View the generated student profile output
